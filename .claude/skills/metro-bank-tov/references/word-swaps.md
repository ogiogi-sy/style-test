## Banking Jargon to Avoid or Explain

These terms are common in banking but confusing to fans. Either replace them or add a brief, plain-English explanation.

| Jargon | Plain English alternative |
|---|---|
| In arrears | Behind on payments |
| Amortisation | How your loan is paid off over time |
| APR | The yearly cost of borrowing (explain in context) |
| Capital | The money you originally borrowed / deposited |
| Collateral | Something valuable you offer as security for a loan |
| Default | When you can't make a payment |
| Equity | The part of your home you own outright |
| Facility | Account, service |
| Fixed rate | A rate that stays the same |
| Gross | Before tax |
| Leverage | (avoid — explain the specific concept instead) |
| Maturity | When your product ends / comes due |
| Net | After tax |
| Portfolio | Your accounts / investments |
| Principal | The original amount borrowed |
| Variable rate | A rate that can go up or down |
| Yield | What you earn on an investment |

## Quick Rules

- If a word has a simpler synonym, use the simpler one
- If you must use a technical term (e.g. for legal/compliance reasons), add a brief explanation in brackets or help text
- When in doubt, read it out loud — if it sounds like a letter from 1995, rewrite it
